# homebrew-lumen

A [Homebrew](https://brew.sh) tap for [Lumen](https://github.com/fzlzjerry/lumen) —
a dynamic, bytecode-compiled programming language with a full toolchain.

## Install

```sh
brew install fzlzjerry/lumen/lumen
```

or

```sh
brew tap fzlzjerry/lumen
brew install lumen
```

`Formula/lumen.rb` is regenerated automatically on every Lumen release by the
release pipeline in the main repo. Do not edit it by hand.
